# Hakintosh-XiaoXinAir14Plus2021ACN
EFI for 小新Air 14 Plus 2021 ACN/IdeaPad 5 Pro 14 ACN

## 正常工作：
- CPU变频
- WIFI
- 电池
- 声音（内外放）
- HDMI外接（高刷）
- 摄像头
- 触控板
## 不完美工作：
- GPU（vega7）支持metal硬件加速，不支持解码，Chromium内核软件会花屏，需要用GLFriend运行
- 蓝牙兼容性不好，如需要输入数字确认的蓝牙键盘无法连接
- USB口2A2C只工作1A1C
- Airdrop只能发现设备不能传输

## 不工作：
- 内置麦克风
